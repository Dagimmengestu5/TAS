# Droga Group Hub (TAS)

A state-of-the-art Talent Acquisition & HR Management System designed for strategic recruitment and operational efficiency.

## 🚀 Core Features
- **Dynamic TA Pipeline**: Real-time tracking of candidate progression from submission to offer.
- **Strategic Approval Layers**: Multi-tier oversight for HR and Executive leadership.
- **Intelligent Job Board**: Automated sorting and high-priority signaling for applicants.
- **Encrypted Transmission**: Secure handling of professional credentials and JD documentation.

## 🛠 Tech Stack
- **Frontend**: React.js with Framer Motion and Lucide Icons.
- **Backend**: Laravel PHP Core with RESTful API architecture.
- **Storage**: Highly scalable asset management for resumes and job specs.

## 📂 Project Structure
- `/frontend`: User interface and interactive components.
- `/app`: Backend controllers, models, and business logic.
- `/database`: Migrations and structural blueprints.
